# Weather App

Today I built a weather app that calls a weather API and stores the data in an array.

The app is styled using Tailwind for my first time doing so. I think it turned out quite well!

This is a learning project that demonstrates my skills in calling and using data from different APIs.

The app is fast, and you can check the weather anywhere in the world.

The app provides the following information:

Temperature
Feels-like temperature
Wind speed
Humidity
Pressure
Weather conditions
Description of the weather
